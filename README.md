#  Practice Tracker App

A clean and simple **music practice tracking app** built with **React Native (Expo)**.

Track your daily sessions, stay consistent, and build better practice habits.

---

## Features

* Log practice sessions (instrument + duration)
*  Automatically grouped by:

  * **Today**
  * **Yesterday**
  * Previous dates
* Edit sessions
*  Delete sessions
*  Persistent storage using AsyncStorage
*  Clean dark UI
*  Built with Expo

---

##  What It Does

* Tap the **+ button**
* Select an instrument (Guitar, Piano, Drums, Singing)
* Enter minutes practiced
* Save
* View sessions grouped by date
* Edit or delete anytime

---

## Tech Stack

* React Native
* Expo
* TypeScript

This app demonstrates:
